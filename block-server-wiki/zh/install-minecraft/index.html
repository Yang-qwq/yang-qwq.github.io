判断你是否安装了java ​

WARNING

这篇教程并不面向Java开发者以及从业人员，如有需要您应考虑查看更专业的指南

论玩MC的第一步是什么，那当然是装运行时环境了

如果你使用的是官方启动器或者PCL2这类会提醒你安装运行环境的，一般都引导你/帮你自动装好，你可以直接跳过这一步

对于手机玩家，一般启动器安装包内也会自带Java环境，您也可以选择跳过这一步

否则的话，请跟着教程继续

测试Java环境 ​

以下假设您的电脑是Windows操作系统

打开运行（快捷键Windows+R，如图），并输入cmd后回车

shell
cmd

接下来你会看到一个黑色窗口（不同电脑可能看到的窗口不同），你需要在里面输入命令后回车执行（如图，但为了简化接下来的步骤，这里选择用代码框代表终端）

然后请输入如下内容：

shell
java -version

表现为：

text
Microsoft Windows [版本 10.0.19045.3324]
(c) Microsoft Corporation。保留所有权利。

C:\Users\yang_>java -version

如果你的输出类似于如下内容，则证明你已经安装了Java环境，可以跳过这一步教程了

text
C:\Users\yang_>java -version
openjdk version "17.0.7" 2023-04-18 LTS
OpenJDK Runtime Environment Microsoft-7626293 (build 17.0.7+7-LTS)
OpenJDK 64-Bit Server VM Microsoft-7626293 (build 17.0.7+7-LTS, mixed mode, sharing)

C:\Users\yang_>

但是如果你的终端提示如下内容，则你需要安装特定版本的Java才可进行游戏哦~
